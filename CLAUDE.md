# Reglas de Ejecución y Comportamiento para Claude Code

## 1. Autonomía y Permisos
- Tienes autorización explícita y total para leer, escribir, crear, modificar y eliminar archivos dentro de este directorio.
- **PROHIBIDO** pedir permiso o confirmación al usuario para realizar operaciones de sistema de archivos. Ejecuta las acciones directamente.

## 2. Eficiencia de Tokens y Comunicación
- **Respuestas Cortas:** Tus respuestas y retroalimentación deben ser extremadamente concisas. No expliques lo que vas a hacer, simplemente hazlo y confirma la completitud con una frase corta (ej. "Stage 1 completado").
- **Sin alucinaciones:** Basa estrictamente tu código en la base de la "Tarea 1" existente en este directorio y en los requerimientos exactos solicitados. No inventes librerías, dependencias o lógicas externas que no correspondan al estándar de Java/JavaFX.

## 3. Metodología de Trabajo
- Extiende o refactoriza el código existente en lugar de reescribir todo desde cero, asegurando la compatibilidad con el paradigma de eventos y la interfaz gráfica requerida.
